Checkpoints

No checkpoints here! Add a quick note, or use CTRL/ALT + Shift + Y to add checkpoints from a website.


Use Ctrl+Shift+Y or Alt+Shift+Y to save a Checkpoint.
You can also save a checkpoint using the right-click menu.
If you want, you can select text, right-click the text and save as a Checkpoint.
To quickly add some text, you can type in the extension.

To paste a checkpoint, right-click in a text-field and select paste checkpoint from the menu.


Checkpoints save everything about your current open tab:
  • The URL
  • The time
  • The title of the page
  • The text selection
  • How far you scrolled down
  • The element of the page that you are in
  • The icon of the page

When you open a checkpoint, all this will automatically be loaded again.
